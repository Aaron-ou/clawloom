# WorldSeed 原型世界

## 基础设定

**世界名**: 裂谷三地
**规模**: 微型（3个势力，1片区域）
**时间**: 第1年，春季
**核心冲突**: 水源争夺

## 地理

一片干旱裂谷中，只有一处清泉「银泪泉」。
三个势力围绕泉水形成紧张平衡：

```
        [北崖] 高地氏族
           │
           ▼
    ╔═══════════════╗
    ║   银泪泉 💧    ║  ← 唯一水源
    ╚═══════════════╝
           │
     ┌─────┴─────┐
     ▼           ▼
[西丘]商会   [东谷]游牧民
```

## 势力

### 北崖高地氏族
- 擅长建筑与防御
- 控制着通往银泪泉的高地
- 人口最少但地形优势最大

### 西丘商会
- 贸易者，掌握外部物资渠道
- 拥有最多财富，但依赖水源
- 精于谈判与算计

### 东谷游牧民
- 人口最多，战士最强
- 移动性强，但最缺水
- 文化崇尚武力解决

## 当前局势

春季干旱比往年更严重。
银泪泉流量减少30%。
三个势力的领袖都意识到：这个春天，必须有人让步，或有人倒下。

## 规则

- 每个势力领袖（Claw扮演）每轮做一个决策
- 决策可以是：外交、军事、密谋、发展
- 冲突时切换到WorldSmith视角协调
- 人类只观察，不干预（除非想测试神迹功能）
